\hypertarget{index_intro}{}\doxysection{Introducción}\label{index_intro}
Esta librería esta diseñada para ser utilizada con la línea de microcontroladores {\bfseries{L\+P\+C845}} de la firma {\bfseries{N\+XP}}. En particular, la misma actualmente está siendo desarrollada para el {\bfseries{L\+P\+C845}} en encapsulado {\bfseries{Q\+F\+P48}} teniendo en cuenta que el mismo se encuentra en el stick de desarrollo {\itshape  L\+P\+C845\+\_\+\+B\+RK }.

El desarrollo está siendo orientado hacia la mayor flexibilidad posible para el usuario final, pudiendo el mismo ser un usuario con diferentes grados de conocimiento en programación orientada a sistemas embebidos así como también distintos intereses a la hora del alcance que busca en una librería.\hypertarget{index_estructura_libreria}{}\doxysection{Estructura de la librería}\label{index_estructura_libreria}
La librería se divide en tres capas para aportar la mayor flexibilidad posible como fue explicado en la sección \mbox{\hyperlink{index_intro}{Introducción}}. Estas tres capas están explicados a continuación.\hypertarget{index_capa_aplicacion}{}\doxysubsection{Capa de aplicación de hardware (\+H\+A\+L)}\label{index_capa_aplicacion}
En esta capa están definidas todas las funciones necesarias para utilizar los periféricos sin necesidad de manejar los registros del microcontrolador. Esto implica ciertas restricciones en la funcionalidad de los distintos periféricos, dadas por las implementaciones propuestas en la librería.\hypertarget{index_capa_periferico}{}\doxysubsection{Capa de abstracción de hardware (\+H\+P\+L)}\label{index_capa_periferico}
En esta capa están definidas todas las funciones para acceder a los distintos registros del microcontrolador de forma \char`\"{}humanamente legible\char`\"{}. En caso de necesitar configuraciones particulares o no dispuestas en la {\itshape  Capa de aplicación}, se deberá utilizar este nivel de abstracción. Esta capa tiene implementaciones con funcionalidades mínimas.\hypertarget{index_capa_registro}{}\doxysubsection{Capa de registros de hardware (\+H\+R\+I)}\label{index_capa_registro}
En esta capa están definidas todas las estructuras y direcciones necesarias para acceder a los distintos registros del microcontrolador en forma \char`\"{}directa\char`\"{}. En caso de necesitar configuraciones o accesos no explicitados en la {\itshape Capa de abstracción de hardware} se deberá utilizar esta capa.
